# 祭品动画图片 | Sacrifice Animation Images

将祭品图片放入此文件夹，用于祭祀按钮的动画效果。  
*Place sacrifice images in this folder for the memorial button animation effects.*

---

## 图片要求 | Image Requirements

| 项目 | Item | 要求 | Requirement |
|------|------|------|-------------|
| 格式 | Format | PNG（推荐透明背景 | *PNG with transparent background recommended*） |
| 尺寸 | Size | 约 100x100 像素 | ~100x100 pixels |
| 命名 | Naming | `examplePic1.png`, `examplePic2.png`, `examplePic3.png` | `examplePic1.png`, `examplePic2.png`, `examplePic3.png` |
| 数量 | Quantity | 至少 1 张，建议 3 张以上以获得更好的随机效果 | At least 1, 3+ recommended for better randomization |

---

## 图片示例 | Image Ideas

可以是 | Can be：
- 🌸 花朵（菊花、百合等）| Flowers (chrysanthemums, lilies, etc.)
- 🕯️ 蜡烛 | Candles
- 🍎 水果 | Fruits
- 🍚 食物 | Food
- 💐 花束 | Bouquets
- 任何具有纪念意义的物品 | Any item with memorial significance

---

## 配置 | Configuration

如需修改图片文件名，请编辑 `assets/js/sacrifice.js` 中的 `imageNames` 数组：  
*To modify image filenames, edit the `imageNames` array in `assets/js/sacrifice.js`:*

```javascript
imageNames: ["yourPic1.png", "yourPic2.png", "yourPic3.png"]
```
