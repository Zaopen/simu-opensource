# 祭品动画图片

将祭品图片放入此文件夹，用于祭祀按钮的动画效果。

## 图片要求

- **格式**: PNG（推荐透明背景）
- **尺寸**: 约 100x100 像素
- **命名**: `examplePic1.png`, `examplePic2.png`, `examplePic3.png`
- **数量**: 至少 1 张，建议 3 张以上以获得更好的随机效果

## 图片示例

可以是：
- 🌸 花朵（菊花、百合等）
- 🕯️ 蜡烛
- 🍎 水果
- 🍚 食物
- 💐 花束
- 任何具有纪念意义的物品

## 配置

如需修改图片文件名，请编辑 `assets/js/sacrifice.js` 中的 `imageNames` 数组：

```javascript
imageNames: ["yourPic1.png", "yourPic2.png", "yourPic3.png"]
```
