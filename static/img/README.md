# 纪念照片 | Memorial Photos

将故人照片放入此文件夹。  
*Place memorial photos of the deceased in this folder.*

---

## 文件命名规则 | File Naming Convention

照片文件名必须与 `layouts/` 目录下的 HTML 文件名一致。  
*Photo filenames must match the HTML filenames in the `layouts/` directory.*

**示例 | Examples：**
- `layouts/demo.html` → `static/img/demo.png`
- `layouts/grandpa.html` → `static/img/grandpa.jpg`

---

## 支持的格式 | Supported Formats

按优先级排序 | Listed in priority order：
1. `.png`
2. `.jpg`
3. `.jpeg`
4. `.gif`
5. `.webp`

---

## 图片建议 | Image Recommendations

| 项目 | Item | 建议 | Recommendation |
|------|------|------|----------------|
| 尺寸 | Size | 建议高度 350-400 像素 | Recommended height: 350-400 pixels |
| 比例 | Aspect ratio | 竖版照片效果最佳 | Portrait orientation works best |
| 风格 | Style | 庄重、温馨的照片 | Solemn and warm photos |

---

## 隐私提示 | Privacy Notice

⚠️ 如果项目托管在公开仓库，请确保您有权使用并公开这些照片。  
*If the project is hosted in a public repository, please ensure you have the right to use and share these photos.*
