# 背景音乐

将背景音乐文件放入此文件夹。

## 支持的格式

- `.mp3`（推荐，兼容性最好）
- `.ogg`
- `.wav`

## 配置方法

在 `layouts/yourpage.html` 中修改音乐路径：

```html
<!-- 本地音乐地址 -->
<p id="mURL" style="display: none;">../static/music/your-music.mp3</p>
```

## 音乐建议

- 选择舒缓、庄重的音乐
- 文件大小建议控制在 5MB 以内，以加快页面加载
- 推荐使用纯音乐或轻音乐

## 版权提示

⚠️ 请确保使用的音乐具有合法授权，或使用无版权音乐（如 Creative Commons 许可）。

## 推荐无版权音乐来源

- [Free Music Archive](https://freemusicarchive.org/)
- [Incompetech](https://incompetech.com/)
- [Musopen](https://musopen.org/)
