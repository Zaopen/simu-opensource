# 背景音乐 | Background Music

将背景音乐文件放入此文件夹。  
*Place background music files in this folder.*

---

## 支持的格式 | Supported Formats

- `.mp3`（推荐，兼容性最好 | *Recommended, best compatibility*）
- `.ogg`
- `.wav`

---

## 配置方法 | Configuration

在 `layouts/yourpage.html` 中修改音乐路径：  
*Modify the music path in `layouts/yourpage.html`:*

```html
<!-- 本地音乐地址 | Local music path -->
<p id="mURL" style="display: none;">../static/music/your-music.mp3</p>
```

---

## 音乐建议 | Music Recommendations

- 选择舒缓、庄重的音乐  
  *Choose soothing and solemn music*
- 文件大小建议控制在 5MB 以内，以加快页面加载  
  *Keep file size under 5MB for faster page loading*
- 推荐使用纯音乐或轻音乐  
  *Instrumental or light music is recommended*
